# Screenshot Placeholders

Drop your screenshots in this folder using these names:

- `01-home.png`: landing page
- `02-play.png`: envelope selection screen
- `03-result-form.png`: winner form and bank selection
- `04-admin-config.png`: admin config panel
- `05-admin-qr.png`: admin payout QR details

Tips:
- PNG format is recommended.
- Keep width around 1600px for clean GitHub rendering.
- If screenshots include private data, blur device IDs, phone numbers, and account numbers.
